清空绘图操作是将已经绘制的图像全部清空，对于清空操作可以执行 `Graphics` 中的 `clear` 操作。

当需要清除一个对象的绘图时候，我们可以执行下面语句

{% highlight java linenos %}
shp.graphics.clear();
{% endhighlight %}

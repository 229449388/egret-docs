### 概述
   
startserver 命令用于开启一个本地服务器
    
------
     
### 使用方法
    
命令原型：    
{% highlight PowerShell %}
egret startserver [project_name] [--port 3000] [-ip] [-serveronly]
{% endhighlight %}
      
-----
    
### 参数说明
       
{% highlight java %}
--port           指定端口号
-ip              是否使用本机IP
-serveronly      是否只运行服务器
{% endhighlight %}

### 概述
   
create_manifest 命令用于在工程目录下生成 manifest.json 清单文件
    
------
     
### 使用方法

命令原型：    
{% highlight PowerShell %}
egret create_manifest { your_project }
{% endhighlight %}
      
-----
    
### 参数说明
    
{% highlight java %}
-all     生成所有文件的清单,若不指定则只生成文档类有引用到的类清单
{% endhighlight %}

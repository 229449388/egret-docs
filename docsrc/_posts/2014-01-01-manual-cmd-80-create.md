### 概述
   
create 命令用于创建一个新的 egret 项目
    
------
     
### 使用方法

命令原型：    
{% highlight PowerShell %}
egret create [project_name] [--type core|gui] [--runtime html5|native]
{% endhighlight %}
      
-----
    
### 参数说明
    
{% highlight java %}
--type       要创建的项目类型 core 或 gui，默认值为 core
--runtime    设置构建方式为 html5 或者是 native 方式，默认值为 html5
{% endhighlight %}

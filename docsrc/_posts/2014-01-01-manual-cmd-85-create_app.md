### 概述
   
create_app 命令用于生成对应的原生项目
    
------
     
### 使用方法

命令原型：    
{% highlight PowerShell %}
egret create_app [app_name] -f [h5_game_path] -t [template_path]
{% endhighlight %}
      
-----
    
### 参数说明
    
{% highlight java %}
-f    html5 项目路径
-t    模板路径，对应egret-andriod-support 或者 egret-ios-support 路径
{% endhighlight %}
